# AI Resume Analyser

An AI-powered Resume Analyser built using Flask, Groq AI, SQLAlchemy, and TiDB.

## Features

- User Login & Signup
- Resume Upload (PDF/DOCX)
- Resume Skill Extraction
- Missing Skill Detection
- AI Roadmap Generation
- Interview Questions
- Resume History
- Secure Login
- Cloud Database (TiDB)

## Tech Stack

- Python
- Flask
- SQLAlchemy
- TiDB
- Groq API
- HTML
- CSS
- JavaScript

## Installation

```bash
git clone https://github.com/KenilVegad/AI_Resume_Analyser.git
cd AI_Resume_Analyser
pip install -r requirements.txt
python app.py
```
